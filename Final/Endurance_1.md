# Endurance

Enhorabuena, habéis sido escogidos para crear el sistema informático de la nave Endurance, una nave interplanetaria creada por la NASA como parte del Proyecto Endurance. Su misión principal será rescatar a los científicos abandonados en misiones previas. 

Puesto que la finalidad principal del sistema es realizar un viaje de largo recorrido es de vital importancia que el sistema sea lo más seguro posible y no tenga ningún fallo. Para asegurarlo deberéis acometer las siguientes tareas.

## DHCP

Implementa dos redes internas usando máquinas virtuales e implementa y configura un servidor DHCP en ambas. 

Una de ellas deberá tener Sistemas Operativos Windows mientras que la otra deberá tener Linux.

Define *MAC Filtering* y *MAC snooping*, que ventajas tienen, si tuviesen alguna desventaja, indica posibles casos de uso y razona si su uso es apropiado para este sistema.

En las redes creadas previamente configura el filtrado MAC.

## Web

Crea un sitio web seguro para la Endurance. 
Se evaluará:

1. La instalación del servicio SSH.
2. La instalación del servidor Web.
3. Las medidas de seguridad implementadas.

Muestra los datos del certificado digital del sitio web
