# Endurance II

Vuestro trabajo ha sido ejemplar hasta ahora. Para la segunda parte deberéis trabajar en el servicio de DNS y Mail.

## DNS

Define *DNSSEC* y *DoH*, que ventajas tienen, si tuviesen alguna desventaja, indica posibles casos de uso y razona si su uso es apropiado para este sistema.

¿Que riesgos puede haber si no actualizamos frecuentemente nuestro servidor DNS? ¿Y si permitimos la resolución recursiva?

En las redes creadas previamente configura el filtrado MAC.

## Mail

Define por que es importante tener en cuenta la seguridad en un servicio de correo electrónico y define otras estrategias para mejorar la seguridad.

Instala un servidor de correo e implementa SMTP Auth.

## FTP

Implementa un servicio FTP en el servidor para poder enviar ficheros.

## Mejora de Servidor Web

Implementa un foro utilizando Wordpress y actualiza el servicio mail para que utilice Roundcube.

Adicionalmente asegúrate que cumples las recomendaciones de seguridad en tu servicio web.
